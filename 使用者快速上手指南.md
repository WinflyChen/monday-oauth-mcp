# 使用者快速上手指南

透過 Telegram，用自然語言操作你的 monday.com 工作區。

---

## 開始之前，你需要準備

- 一個 **Telegram 帳號**（手機或電腦版皆可）
- 一個 **Monday.com 帳號**（需要有實際資料的帳號）

---

## 安裝 Telegram

如果你還沒有安裝 Telegram，請先下載：

| 平台 | 下載連結 |
|------|----------|
| 📱 iPhone / iPad | [App Store 下載](https://apps.apple.com/app/telegram-messenger/id686449807) |
| 🤖 Android | [Google Play 下載](https://play.google.com/store/apps/details?id=org.telegram.messenger) |
| 💻 Mac | [Mac App Store 下載](https://apps.apple.com/app/telegram/id747648890) |
| 🖥️ Windows | [Windows 下載](https://desktop.telegram.org/) |
| 🌐 網頁版 | [web.telegram.org](https://web.telegram.org/) |

安裝完成後，用手機號碼註冊帳號即可。

---

## 步驟一：找到 Bot

在 Telegram 搜尋欄輸入：

```
@igroup_kevin_bot
```

點擊進入對話。

---

## 步驟二：啟動 Bot

發送以下訊息：

```
/start
```

Bot 會歡迎你，並提示你需要先登入才能使用。

---

## 步驟三：連結你的 monday.com 帳號

發送以下訊息：

```
/login
```

Bot 會回傳一個**授權連結**，點擊後會跳轉到 monday.com 的授權頁面。

> 📌 這個步驟只需要做**一次**。之後每次使用不需要重新登入。

---

## 步驟四：在 monday.com 授權

### 💻 電腦版

1. 點擊 Bot 傳來的連結，頁面會直接開啟 monday.com 授權
2. 如果尚未登入，先用帳號密碼登入
3. 看到「Authorize」或「允許」按鈕，點擊確認
4. 頁面顯示「Authorization Successful」代表成功了
5. 回到 Telegram，Bot 會通知你已完成連結

### 📱 手機版（重要）

monday.com 會偵測手機瀏覽器並強制導向 App 下載頁，請依下列方式處理：

**Android Chrome：**
1. 點擊 Bot 的連結，進入授權頁面後點「前往 monday.com 授權」
2. 若出現「Open in Browser」對話框，勾選 **「Always open links from auth.monday.com in browser」** 再點 Continue
3. 被導到下載頁後，點右上角 **⋮** 選單
4. 選擇 **「Open in...」**，然後啟用「桌面版網站（Desktop site）」
5. 重新整理，即可看到 monday.com 授權登入表單

**iOS Safari：**
1. 點擊 Bot 的連結，進入授權頁面後點「前往 monday.com 授權」
2. 被導到下載頁後，點右下角 **↑↓** 分享按鈕
3. 選擇 **「Open in...」**，然後啟用「以桌面版方式查看網站」
4. 重新整理，即可看到 monday.com 授權登入表單

> 💡 啟用桌面模式後，monday.com 會把你的手機瀏覽器當成桌面設備，就能正常顯示授權頁面。

---

## 步驟五：開始使用

授權完成後，你可以直接用中文跟 Bot 說話，例如：

| 你說的話 | Bot 會做的事 |
|---------|------------|
| 幫我列出我的所有 board | 顯示你的 monday.com 工作板清單 |
| 建立一個叫「行銷企劃」的 board | 在你的工作區新增一個 board |
| 我的板子有哪些 | 列出所有 board 名稱與連結 |

---

## 常用指令

| 指令 | 用途 |
|------|------|
| `/start` | 啟動 Bot |
| `/login` | 連結 monday.com 帳號（首次使用或重新授權） |
| `/status` | 查看目前的登入狀態 |
| `/reset` | 清除登入狀態（如需切換帳號） |

---

## 常見問題

**Q：點授權連結出現「無法連上這個網站」？**  
A：連結可能已過期，重新發送 `/login` 取得新的連結。

**Q：Bot 沒有回應我的訊息？**  
A：先發送 `/status` 確認是否已登入。若未登入，重新執行 `/login`。

**Q：我可以在多個裝置使用嗎？**  
A：可以。只要在同一個 Telegram 帳號下都能使用，授權只需做一次。

**Q：手機點連結一直跑到 App 下載頁怎麼辦？**  
A：這是 monday.com 的行動端偵測機制。請在手機瀏覽器的選單中啟用「桌面版網站」模式，詳細步驟請看步驟四的手機版說明。

**Q：我的 monday.com 資料安全嗎？**  
A：Bot 只會使用你授權的權限讀取/操作資料，不會儲存你的帳號密碼。
