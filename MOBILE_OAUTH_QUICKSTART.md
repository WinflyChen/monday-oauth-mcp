# 行動版授權完整指南 v2.0

## 🎯 新功能：自動行動設備檢測 + 逐步引導

現在系統會自動偵測您使用的是行動設備還是桌面設備，並提供對應的授權方式！

---

## 📱 行動用戶授權流程

### 第 1 步：在 Telegram 中執行 `/login`

```
/login
```

您會看到一個授權按鈕：

```
🌐 前往授權頁面    ← 點擊此按鈕
📋 複製授權連結    ← 或複製連結
```

### 第 2 步：點擊「前往授權頁面」

系統會在外部瀏覽器（Chrome/Safari）中打開頁面，並自動偵測您的設備。

**如果您使用行動設備：**
- 服務器會先顯示一個**指導頁面**
- 頁面會說明如何使用「Open in...」或「桌面模式」
- 頁面提供詳細的 4 個步驟說明

**如果您使用桌面電腦：**
- 直接顯示 Monday.com 授權頁面
- 輸入帳號並授權即可

### 第 3 步：按照提示完成授權

**對於行動用戶，指導頁面會告訴您：**

1. ✅ 點擊「前往 Monday.com 授權」按鈕
2. ✅ 看到下載頁面後，點擊右上角的「⋯」(Chrome) 或右下角的「↑↓」(Safari)
3. ✅ 勾選「桌面版網站」(Chrome) 或「以桌面版方式查看網站」(Safari)  
4. ✅ 重新整理頁面，應該就看到授權表單了
5. ✅ 輸入 Monday.com 帳號密碼並授權

### 第 4 步：授權完成

授權成功後，會自動回傳 Telegram 並顯示確認訊息。

## 🖥️ 桌面用戶授權流程

### 簡單三步：

1. 在 Telegram 中執行 `/login`
2. 點擊「前往授權頁面」按鈕
3. 直接看到 Monday.com 授權表單，輸入帳號並授權

✅ 完成！

---

## 🆘 備用方案：複製授權連結

如果按鈕無法使用，您可以手動複製連結：

### Android 用戶

1. 在 Telegram 點擊 `/login`
2. 點擊「📋 複製授權連結」按鈕
3. 系統會發送一個可複製的連結
4. 複製該連結
5. 打開 Chrome
6. 在位址列粘貼並按 Enter
7. **重要**：看到下載頁面後，點擊右上角「⋯」
8. 向下滑，勾選「**桌面版網站**」✓
9. 重新整理頁面
10. ✅ 應該看到授權表單，完成授權

### iPhone/iPad 用戶

1. 在 Telegram 點擊 `/login`
2. 點擊「📋 複製授權連結」按鈕
3. 打開 **Safari**（不是 Chrome）
4. 在位址列粘貼連結並按 Enter
5. 點擊右下角「↑↓」按鈕
6. 向上滑，勾選「**以桌面版方式查看網站**」
7. 重新整理頁面
8. ✅ 應該看到授權表單，完成授權

## ✅ 常見問題

### Q1：為什麼我的行動設備被導向下載頁面？

**A：** 這是 Monday.com 的正常行為。他們希望鼓勵行動用戶使用官方應用程式。我們的解決方案是通過啟用瀏覽器的「桌面模式」，來告訴 Monday.com「我想使用網頁版」。

### Q2：「桌面模式」會影響我的手機嗎？

**A：** 不會！「桌面模式」只是告訴網站把您當作桌面用戶對待。您的手機不會有任何改變。完成授權後，您可以關閉此設定，手機會恢復正常。

### Q3：我在 iPhone 上試過了，但仍然看到下載頁面？

**A：** 請確認您有正確勾選「以桌面版方式查看網站」，並且已經**重新整理**了頁面。有時需要等待 2-3 秒頁面才會載入。

### Q4：授權後，我可以在 Telegram 中直接使用嗎？

**A：** 是的！授權完成後，Telegram Bot 會自動為您準備好一切。您可以直接使用以下命令：
- `/boards` - 列出 Monday 看板
- `/status` - 查看授權狀態
- 直接輸入自然語言命令建立任務

### Q5：複製連結失敗怎麼辦？

**A：** 
1. 檢查網路連線
2. 重新執行 `/login` 命令
3. 嘗試使用不同的瀏覽器
4. 如果持續失敗，嘗試使用無痕/隱私模式

## 🎓 技術背景

### 為什麼需要「桌面模式」？

```
情況 1：直接訪問（不使用桌面模式）
─────────────────────────────────────
您的 Chrome（行動版）
    ↓ User-Agent: Android 13
Monday.com 檢測到行動設備
    ↓
導向 App Store 下載

情況 2：使用「桌面模式」
─────────────────────────────────────
您的 Chrome 啟用「桌面模式」
    ↓ User-Agent: Windows 10
Monday.com 檢測到桌面設備
    ↓
返回授權表單 ✅
```

### 伺服器端做了什麼？

1. **設備智能偵測** - 檢查 HTTP User-Agent 頭
2. **條件重定向** - 行動設備→指導頁面，桌面→授權頁面
3. **備選機制** - 提供複製連結選項
