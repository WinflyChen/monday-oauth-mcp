# Android 「Always Open」設定完整指南

## 🎯 目標

第一次授權時勾選「Always open links from auth.monday.com in browser」，這樣下次 Telegram 機器人發送授權連結時，就會**自動在 Chrome 中打開**，不用再次選擇瀏覽器。

---

## 📱 第一次授權流程

### 步驟 1：執行 `/login` 後點擊「前往授權」

在 Telegram 執行命令：
```
/login
```

點擊「🚀 前往 Monday.com 授權」按鈕。

### 步驟 2：看到「Open in Browser」對話框

您會看到這個對話框：

```
┌─────────────────────────────────────────┐
│ Open in Browser                         │
│                                         │
│ Do you want to open this link in your  │
│ default browser?                        │
│                                         │
│ ☐ Always open links from              │
│   auth.monday.com in browser            │
│                                         │
│         [Cancel]      [Continue]       │
└─────────────────────────────────────────┘
```

### **🔴 重要：一定要打勾！**

**在點擊「Continue」之前**，點擊左邊的方框 ☐ 使其變為 ☑️

```
☐ → ☑️  (打勾)
```

### 步驟 3：點擊「Continue」

現在點擊「Continue」按鈕，開始授權流程。

### 步驟 4：完成授權

按照指導頁面的 4 個步驟完成授權（使用桌面模式等）。

---

## ✅ 第二次及以後

**下次您再在 Telegram 中點擊授權連結時：**

- ✅ **自動在 Chrome 中打開**（無需選擇瀏覽器）
- ✅ **跳過「Open in Browser」對話框**
- ✅ **直接進入授權流程**

---

## 🆘 如何取消「Always Open」設定

如果您想要取消這個設定（使其回到每次都詢問）：

### Android 設定方法

1. 打開手機的「設定」應用
2. 進入「應用程式」或「應用程式管理器」
3. 找到「Google Chrome」或「Chrome」
4. 點擊進入
5. 向下滑找到「預設應用程式」
6. 點擊「清除預設值」按鈕
7. ✅ 完成！下次會重新詢問

### 快速路徑（某些 Android 版本）

設定 → 應用程式 → Chrome → 權限 → 清除 URL handling 預設值

## 🔧 進階設定

### 設定「Always Open」後的行為

**通常的行為：**
- Monday.com 授權相關連結 → 自動 Chrome 打開

**其他連結怎麼辦？**
- 只影響來自 `auth.monday.com` 的連結
- 其他網站的連結行為不變

### 如何針對不同網域設定不同應用

Android 沒有直接 UI 做這個，但 Chrome 會記錄每個網域的預設應用。

如果您想使用不同的瀏覽器（例如 Firefox）打開其他連結，可以：

1. 清除 Chrome 的預設值（如上所述）
2. 打開 Firefox
3. 類似地設定 Firefox 為預設
4. 系統會詢問您要為哪些連結使用 Firefox

---

## 📋 常見問題

### Q1：打勾會影響其他應用嗎？

**A：** 不會。只會影響 `auth.monday.com` 這個特定網域的行為。其他網站的連結不受影響。

### Q2：我不小心打勾了，想要取消怎麼辦？

**A：** 沒問題。按照上面「如何取消」部分的步驟操作即可。

### Q3：我在 iPhone 上應該怎麼做？

**A：** iPhone 上 Safari 會自動記住您的選擇，無需特別設定。只需在第一次授權時選擇「在 Safari 中打開」即可。

### Q4：打勾後，如果我想用其他瀏覽器打開怎麼辦？

**A：** 
1. 清除預設值（按照上述步驟）
2. 下次會重新詢問，選擇您想用的瀏覽器
3. 再次「Always Open」（可以選擇新瀏覽器）

### Q5：這個設定會永久保存嗎？

**A：** 
- ✅ 是的，會保存到您清除為止
- ✅ 即使重啟手機也會保留
- ✅ 除非卸載或清除 Chrome 數據
